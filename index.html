5C APIs

5C Courses Public API

This is an API which allows you to query for all courses offered at the Claremont Colleges (5C). The base URL is:

http://course-api.herokuapp.com/$DEPARTMENT_ID/$COURSE_ID/$SECTION_ID

The return format is JSON. Possible path parameters are:

Path Example parameters Description
department MATH, ECON, GOVT, CSCI Returns a JSON array containing all courses offered in that Department. A list of proper Department ID's are located at the root URL here. Generally they are a sensible 4-letter abbreviation of a Department.
course number 30, 50, 20, 5 Returns a JSON array containing all sections offered of that course. In general, you can figure out all the proper course numbers for a given department by first looking at a Department, and iterating over its course values.
section number 1, 2, 3 Returns a JSON array containing all courses in a particular department with that section number. In a sane world, this is a 1-element list, but the kind Registrars give us exceptions to that rule.

Finally, each individual JSON object returned by any of the above API calls has the following format:

Key Example value
reg_limit "18 / 18"
campus "CM"
end "4:00pm"
course "GOVT020"
status "Closed"
hours "1.00"
title "Introduction to American Politics"
section "01"
start "2:45pm"
room "103"
bldg "RN"
instructor "Miller K"
days "MW"

5C Dining Public API

The dining data can be found here:

http://dining-api.herokuapp.com/$DINING_HALL

This API allows you to query the food served by the 6 major dining halls in Claremont. The rteurn format is JSON. You can either request the root and then iterate over all dining halls, or use a single path parameter to specify certain dining halls:

Parameter All possible values
Dining Hall ID collins, pitzer, frary, frank, scripps, mudd, all


The data returned by each route is a nested JSON object with the following format:

Key Example value
"tue" Maps to inner JSON object containing all meals
"dinner" Maps to innermost JSON object containing all dishes served at that meal
